# Salam de Biscuiti (Vegan)

## Ingredients

| Ingredient | Quantity | Notes |
|---|---|---|
| Biscuiti simpli vegani | 920g | Zdrobiti in texturi mixte: firimituri, bucati mici si medii |
| Zahar | 300g | 270g baza + 30g pentru laptele extra; poti merge pana la 310g daca vrei mai dulce |
| Lapte vegetal | 430ml | 320ml baza + 110ml pentru corectarea texturii |
| Unt vegetal / margarina 80% grasime | 285g | 235g baza + 50g extra |
| Cacao | 37g | ~5.5 linguri baza + 1 lingura rasa extra |
| Nuca prajita | 160g | Optional |
| Rahat / stafide / merisoare | 160g | Optional |
| Esenta de rom sau rom | 2 linguri | Optional |
| Sare | 1 varf de cutit | |

## Original proportions (for 860g biscuiti)

| Ingredient | Quantity |
|---|---|
| Biscuiti simpli vegani | 860g |
| Zahar | 280g |
| Lapte vegetal | 400ml |
| Unt vegetal / margarina 80% grasime | 265g |
| Cacao | 35g |
| Nuca prajita | 150g |
| Rahat / stafide / merisoare | 150g |
| Esenta de rom sau rom | 2 linguri |
| Sare | 1 varf de cutit |

## Equipment

- Folie alimentara
- Cratita / ibric

## Method

1. Zdrobeste biscuitii in bucati mixte (firimituri + bucati mici + bucati medii).
2. Incalzeste 320ml lapte vegetal cu 270g zahar, cacao si sare pana se dizolva.
3. Adauga 235g unt vegetal si amesteca pana se topeste complet.
4. Toarna peste biscuiti si amesteca cu nuca, rahat/stafide si rom.
5. **Corectie textura (daca e prea uscata):** incalzeste separat 110ml lapte vegetal cu 30g zahar si 1 lingura rasa cacao. Incorporeaza uniform in compozitie.
6. Topeste separat inca 50g unt vegetal si incorporeaza pentru stabilitate si legare.
7. Ruleaza strans in folie alimentara.
8. Lasa la frigider minim 6-8 ore, ideal peste noapte.

## Textura corecta

- Lipicioasa
- Compacta
- Modelabila
- Nu curge
- Cand o strangi in mana ramane legata

### Troubleshooting

| Problema | Fix |
|---|---|
| Prea uscata | Adauga treptat mix de lapte vegetal cald + zahar + cacao |
| Prea moale | Adauga biscuiti zdrobiti fin, sau lasa mai mult la rece |
| Nu se leaga | Adauga grasime solida topita, nu doar lapte |

## Storage

- La frigider

## Notes

- La varianta vegana, grasimea conteaza mult pentru textura finala
- Uleiul de cocos sau margarina mai grasa stabilizeaza mai bine decat un lapte vegetal subtire
- Nu adauga tot lichidul dintr-o data daca faci ajustari ulterioare
- Reteta fara nuca — textura se testeaza fara nuca
- Scaled up from original (860g biscuiti → 920g biscuiti, factor 1.07x)
